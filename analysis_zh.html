示意图只描述建模与数据整理逻辑，不代表实验配方或制备流程。

结构-活性建模 Structure → Activity
脂质结构 SMILES / 描述符 / ChemBERTa
骨架留出 新 scaffold 测试泛化
活性预测 R² / MAE / Spearman

目标是判断“只看 ionizable lipid 结构”能否预测递送或表达活性，适合做候选排序和特征价值评估。

mRNA 递送子集 Payload-specific subset
载荷筛选 payload = mRNA / reporter
语境保留 cell / tissue / route / readout
递送任务 表达、递送效率、靶向趋势

目标是把 mRNA 相关记录从总库中单独拿出，并保留实验语境，避免把不同载荷和 readout 混成一个标签。

配方空间整理 Formulation space
组分组合 ionizable / helper / sterol / PEG
比例字段 molar ratio / N:P / dose
配方地图 来源覆盖、缺口、相似配方

目标是从“单个脂质”升级到“完整 LNP 配方”视角，用于检索、去重、相似配方查找和后续多模态建模。

物化性质与质量 Quality readouts
配方/工艺上下文 组分、比例、制备批次
颗粒 readout size / PDI / zeta / EE
质量建模 可制造性与稳定性线索

目标是把活性之外的质量指标独立出来，判断哪些配方区域更可能形成合适颗粒并支持稳定递送。
